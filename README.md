# Laboratorio HTML, CSS y JS

Página web con un contador interactivo.

## Requisitos cumplidos

1. Contador visible con estilos personalizados
2. Botón para incrementar el valor
3. Botón para decrementar el valor
4. Cambio de color del contador al llegar a 10

## Archivos

| Archivo | Descripción |
|---------|-------------|
| `index.html` | Estructura de la página |
| `styles.css` | Estilos del contador y botones |
| `script.js` | Lógica del contador |

## Cómo ejecutarlo

Abre el archivo `index.html` en el navegador.

## Uso

- **Incrementar**: suma 1 al contador
- **Decrementar**: resta 1 al contador
- Cuando el valor es 10 o mayor, el número cambia de color
